# 🎓 سیستم جامع مدیریت مکاتب افغانستان - نسخه کامل

## 📋 خلاصه پروژه

سیستم جامع مدیریت مکاتب افغانستان یک پلتفرم مدرن برای مدیریت آموزشی در افغانستان است که با پشتیبانی کامل از زبان‌های محلی و ساختار آموزشی ملی طراحی شده است.

## ✅ امکانات کامل شده

### 🗄️ **دیتابیس و مدل‌ها**
- **AfghanSchool** - اطلاعات کامل مکاتب با کدهای وزارتی
- **AfghanStudent** - دانش‌آموزان با تذکره و اطلاعات خانوادگی  
- **AfghanTeacher** - معلمان با صلاحیت‌های حرفه‌ای
- **AfghanCurriculum** - برنامه درسی ملی افغانستان

### 🔧 **Backend (Node.js + Express)**
- **3 Route کامل** با API های RESTful
- **مدیریت مکاتب** (`/api/afghan-schools`)
- **مدیریت دانش‌آموزان** (`/api/afghan-students`)
- **مدیریت معلمان** (`/api/afghan-teachers`)
- **گزارشات تحلیلی و وزارتی**
- **پشتیبانی از CRUD کامل**
- **فیلترهای پیشرفته و جستجو**

### 🎨 **Frontend (React + Modern UI)**
- **داشبورد مدیریتی کشوری**
- **نقشه تعاملی با Leaflet**
- **مدیریت مکاتب با فیلترها**
- **سیستم گزارشات جامع**
- **رابط کاربری چندزبانه (دری/پشتو)**
- **طراحی واکنش‌گرا و مدرن**

## 🌟 ویژگی‌های کلیدی

### 🌍 **پوشش جغرافیایی کامل**
- **34 ولایت** افغانستان با کدهای استاندارد
- **ولسوالی‌ها و قریه‌ها** برای موقعیت دقیق
- **مختصات جغرافیایی** برای نقشه و مسیریابی

### 📚 **پشتیبانی زبانی**
- **انگلیسی**: برای سیستم و گزارشات
- **دری**: زبان اصلی آموزش در افغانستان
- **پشتو**: زبان دوم رسمی کشور

### 🆔 **اسناد هویتی رسمی**
- **تذکره**: شناسه اصلی شهروندان افغانستان
- **شماره جلد و صفحه**: برای مراجع دقیق
- **گواهینامه معلمی**: برای تایید صلاحیت‌ها

### 📊 **گزارشات وزارتی**
- **گزارش سالانه** با آمار کامل
- **گزارشات ولایتی** تفکیک شده
- **گزارشات جنسیتی** و انواع مکاتب
- **خروجی PDF و Excel**

## 🚀 **دسترسی به سیستم**

### مسیرهای اصلی:
- **داشبورد**: `http://localhost:3000/afghan-dashboard`
- **نقشه مکاتب**: `http://localhost:3000/afghan-map`
- **مدیریت مکاتب**: `http://localhost:3000/afghan-schools`
- **گزارشات**: `http://localhost:3000/afghan-reports`

### API ها:
- **داشبورد**: `GET /api/afghan-schools/dashboard`
- **لیست مکاتب**: `GET /api/afghan-schools`
- **نقشه**: `GET /api/afghan-schools/map-data`
- **گزارش سالانه**: `GET /api/afghan-schools/reports/annual`

## 🔧 **راهنمای توسعه**

### افزودن مکتب جدید:
```javascript
POST /api/afghan-schools
{
  "name": "نام مکتب",
  "nameDari": "نام مکتب به دری",
  "schoolCode": "KBL-003",
  "province": "kabul",
  "schoolType": "primary",
  "ownership": "government"
}
```

### دریافت گزارش سالانه:
```javascript
GET /api/afghan-schools/reports/annual?year=2023
```

### جستجوی مکاتب:
```javascript
GET /api/afghan-schools?province=kabul&schoolType=primary&search=حبیبیه
```
